ตั้งแต่ยุค AI มาถึง... เราก็มีลูกพอดี ผ่ามม ไม่ใช่ละ เอาใหม่ๆ

ตั้งแต่ยุค AI มาถึง ส่วนตัวก็หนีการจ่าย subscription มาตลอด เพราะคิดว่าหาเงินจากมันไม่ได้ก็ถือว่าไม่คุ้ม แต่ก็ใช้หลายๆ ตัวที่มีแบบฟรีให้ใช้ ถ้าหมดโควต้าก็วนๆ ไปเจ้าอื่นเรื่อยๆ

ในยุคแรกๆ ที่ยังไม่มี AI agents ก็ใช้ `GitHub Copilot` มาก่อน ซับรายปี **100 USD** เพราะเบิกบริษัทได้ จัดว่าคุ้ม ซึ่งตอนแรกก็ใช้แค่ copilot suggestions ผสมๆ ไปกับแชทจากทั้ง `ChatGPT`, `Claude`, `Gemini` เพื่อช่วยในการทำงาน ก็อปแปะก็อปวาง double check กับ Stack Overflow บ้าง

จนยุค agent มาถึงอย่างรวดเร็ว ทั้ง `Claude Code` เปิดตัว `Codex` เปิดตาม เราก็ยังไม่ลังเล ไม่จ่ายเพิ่ม เพราะคิดว่าเปลือง แล้ว agent ก็มาใน `VS Code` ก็ใช้ต่อเลย เลือกโมเดลได้ด้วย ของดี ก็ใช้มาเรื่อยๆ ไม่มีปัญหาอะไร เอ้อ เกือบลืม จริงๆ มีช่วงที่ลองเล่นพวก `Cline`, `Roo Code` แต่ก็ไม่ค่อยอิน เพราะงานเรามันแอบยาก และเราก็สั่ง AI ไม่ได้เก่งขนาดนั้น ไม่ใช่สาย prompt engineering สมัยนั้น model ฟรีก็ไม่ได้ฉลาดมาก ลองแค่ agent เขียนงานจุบจิบให้เราได้ก็เจ๋งแล้ว

อะกลับไปที่ใช้ agent ใน `VS Code` ไปเรื่อยๆ จนมาต้นปี 2026 นี่แหละ ที่เริ่มรู้สึกว่า**ไม่ได้เขียนโค้ดเองจริงๆ แล้ว** โดยได้ย้ายมาใช้ `OpenCode` เต็มตัว และแพลน **100 USD** ต่อปีเหมือนจะเริ่มไม่พอ ข้ามปีใหม่มาเดือนสองเดือนแรกก็กระเบียดกระเสียรพยายามใช้ให้พอแต่สุดท้ายก็ทะลุมา extra usage แต่ก็ยัง limit ไว้ที่ **15 USD** ได้ ช่วงนั้นนี่แพลนด้วย `Opus 4.5` เขียนด้วย `Sonnet 4.5` ค่อนข้างเข้ามือเลย

ผ่านมาไม่นาน `GitHub Copilot` ก็น่าจะเริ่มแบกไม่ไหว จนข่าว blog นี้ออกมา — [GitHub Copilot is moving to usage-based billing](https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/) — ก็เริ่มเสียวสันหลังละ ลำพังใช้อยู่นี่ก็ไม่พออยู่ละ ยังจะเพิ่ม multiplier เปลี่ยนแผนคิดเงิน โอ้โห จบกัน ก่อนข่าวออกไม่กี่วันพึ่งขอเบิกค่าซับ `Copilot Pro+` เพราะ Pro มันเริ่มไม่พอทำงานละ พอรู้ข่าวแจ้งบอสเลยว่าสถานการณ์ `GitHub Copilot` น่าจะเป็นงี้ เดี๋ยวน่าจะต้องเบิกเพิ่มนู่นนี่ ซึ่งบอสก็ใช้ `Claude Code` มาก่อนแล้วเพื่อทดลองว่าจะคุ้มมั้ยถ้าซับให้ทั้งบริษัท ปรากฏว่าผ่านมาเดือนเดียว ได้ใช้ `Claude Code Pro Max` เฉยแบบงงๆ ไอเราที่ใช้ `OpenCode` เข้ามือแล้วก็ต้องย้ายถิ่นฐานเพราะบริษัทเลือกแล้ว

และผ่านมาสองสามวันซึ่งก็คือวันนี้พบว่า `Claude Code` มันคือ**กับดักความแพง**อย่างนึง คือแปปเดียวก็ใช้เข้ามือเลย ปิดงานง่ายๆ ได้เร็วกว่า `OpenCode` ประมาณนึง ไม่ต้องบริหารหลาย session ในหลาย tabs UX ก็คือดีประมาณนึงเลย มี plugin `WakaTime` เรียบร้อย เช็คแล้วเวลาทำงานเฉลี่ยต่อวันเพิ่มมานิดหน่อย ไม่รู้ดีหรือไม่ดีแต่ก็ทำงานเพลินดีแหละ อย่าง `/remote-control` ที่ built-in มานี่ก็สะดวกเลย `OpenCode` ไปลองหาลง plugin แล้วก็รู้สึกไม่ค่อยเข้ามือเท่าไหร่

อย่างที่เห็นกันว่า AI coding agents นี่**มาไวไปไวมากกกก** ไม่รู้อนาคตอันใกล้จะเป็นยังไง แต่ก็หวังว่าจะตามทันในระดับนึงให้พอเอาตัวรอดให้ได้ก็ยังดี
